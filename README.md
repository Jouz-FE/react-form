# react-form

![Github Actions](https://github.com/seonghyeonkimm/react-form/workflows/chromatic/badge.svg)

React library to help to build form easier than before

## Requirement

- nvm
- node@lts
- yarn

## Install

```shellscript
yarn install
```

## Storybook

```shellscript
yarn storybook
```

## Test

```shellscript
yarn test
```
